# Security Policy

## Supported Versions

Currently, only the latest version of YourDDO is supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

We take the security of YourDDO seriously. If you believe you have found a security vulnerability, please report it to us by following these steps:

1. **Do not open a public GitHub issue.** This helps prevent the vulnerability from being exploited before a fix is available.
2. Report the issue on the **YourDDO Discord server** by contacting a moderator or developer.
3. Include as much detail as possible, such as steps to reproduce, the potential impact, and any suggested fixes.

### What to expect

- We will acknowledge receipt of your report within 48 hours.
- We will provide an estimated timeline for a fix.
- Once a fix is released, we will give you credit for the discovery (unless you prefer to remain anonymous).

Thank you for helping keep YourDDO safe!
